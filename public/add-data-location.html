Add Data Location

Application ID: The ID of the associated application
Data Location: The location this data is stored at
Path: The path of the data relative to the above location. For example, a folder called "appName" inside a folder "publisher", located the local Application Data folder, would be entered as "publisher\appName"
Folder or Directory: Set to True (1) if the above path is to a folder or a file
