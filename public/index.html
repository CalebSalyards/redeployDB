Welcome to RedeployDB

RedeployDB is an API that provides access to the Redeploy application database. Within you can get all of the data you need to confidently back up data for any Windows application you want, and ensure that you can automatically re-download, install and "redeploy" your data on a fresh installation of Windows, be it on a new device or a freshly reset one.
